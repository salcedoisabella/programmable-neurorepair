# Project Status

## Current stage
Early computational engine with a functioning proof of concept.

## Current scope
- remyelination as first tractable wedge
- oligodendrocyte-lineage state progression
- repair-state signature derivation
- candidate control-layer identification
- transition-oriented intervention logic

## What already exists
- multi-dataset implementation
- mature repair-state program
- candidate control architecture layers
- cross-dataset directional support
- intervention-oriented prioritization logic

## Long-term direction
Programmable Neurorepair is being developed as the first wedge into a broader computational platform for mapping neural state transitions, identifying molecular control architecture, and determining where intervention becomes tractable across the central nervous system.
